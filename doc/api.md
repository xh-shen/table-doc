# API

## STableProvider Props

| 名称              | 类型              | 默认值                                            | 说明                                                                                                                                                                                                                                     | 必传 |
| ----------------- | ----------------- | ------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---- |
| bordered          | Booleaen          | -                                                 | 是否显示表格边框                                                                                                                                                                                                                         | N    |
| hover             | Booleaen          | -                                                 | 是否显示鼠标悬浮状态                                                                                                                                                                                                                     | N    |
| highlightSelected | Booleaen          | -                                                 | 可选择是否显示选中状态                                                                                                                                                                                                                   | N    |
| rowKey            | String / Function | -                                                 | 使用 rowKey 唯一标识一行数据，泛型 T 指表格数据类型。TS 类型：`STableProviderProps<T>['rowKey']`。<a href="./type" target="_blank" rel="noreferrer">类型定义</a>                                                                         | N    |
| size              | String            | 'default'                                         | 表格尺寸。可选项：small/default/large，泛型 T 指表格数据类型。TS 类型：`STableProviderProps<T>['size']`。<a href="./type" target="_blank" rel="noreferrer">类型定义</a>                                                                  | N    |
| locale            | Object            | \{filterTitle: '筛选',filterConfirm: '确定',...\} | 国际化配置，默认简体中文，具体使用查看<a href="../guide/locale" target="_blank" rel="noreferrer">国际化</a>。TS 类型：`STableLocale \| STableProviderProps<T>['locale']`。<a href="./type" target="_blank" rel="noreferrer">类型定义</a> | N    |
| themeColor        | String            | -                                                 | 配置 Table 主题色，仅支持十六进制颜色值，如：`#00b96b`                                                                                                                                                                                   | N    |
| customClass       | String            | -                                                 | 自定义 Table 类名                                                                                                                                                                                                                        | N    |

| allowColumnDrag        | Boolean           | false                                             | 是否允许拖拽列排序，全量开启，单独开启请使用`column.drag`，优先级低于`column.drag`                                                                                                                                                                                                                                  | N    |
| animateRows            | Boolean           | true                                              | 是否开启动画                                                                                                                                                                                                                                                                                                        | N    |

| virtual                | Boolean           | true                                              | 是否开启虚拟滚动，数据条数必须大于`virtualThreshold`设置的伐值，可编辑表格时自动关闭虚拟滚动                                                                                                                                                                                                                        | N    |
| xVirtual               | Boolean           | true                                              | 是否开启横向虚拟滚动，数据条数必须大于`virtualThreshold`设置的伐值，可编辑表格时自动关闭虚拟滚动                                                                                                                                                                                                                    | N    |
| [columns](#Column)     | Array             | []                                                | 列配置，泛型 T 指表格数据类型。TS 类型：`Array<STableColumnType<T>> \| STableColumnsType<T> \| STableProps<T>['columns']`。                                                                                                                                                                                         | N    |
| childrenColumnName     | String            | 'children'                                        | 指定树形结构的列名                                                                                                                                                                                                                                                                                                  | N    |
| dataSource             | Array             | []                                                | 数据源，泛型 T 指表格数据类型。TS 类型：`Array<T>`                                                                                                                                                                                                                                                                  | N    |
| rowHeight              | Number / Function | 47                                                | 配置行高，组件内部默认会根据 `size` 自动调整高度，如果需要自定义高度可使用该属性，`size='small'`时`rowHeight=39`；`size='large'`时`rowHeight=55`。TS 类型：`STableProps<T>['rowHeight']`。<a href="./type" target="_blank" rel="noreferrer">类型定义</a>                                                            | N    |
| headerHeight           | Number / Array    | 47                                                | 表头高度，组件内部默认会根据 `size` 自动调整高度，如果需要自定义高度可使用该属性，表格分组时，可以使用数组的方式设置不同层级的高度，`size='small'`时`headerHeight=39`；`size='large'`时`headerHeight=55`。TS 类型：`STableProps<T>['headerHeight']`。<a href="./type" target="_blank" rel="noreferrer">类型定义</a> | N    |
| autoHeaderHeight       | Boolean           | false                                             | 是否开启自动表头高度，开启后会全量加载表头部分                                                                                                                                                                                                                                                                      | N    |
| height                 | Number / String   | -                                                 | 表格高度，超出后会出现滚动条。示例：`100, '300'`。值为数字类型，会自动加上单位 px。如果不是绝对固定表格高度，建议使用 `maxHeight`                                                                                                                                                                                   | N    |
| maxHeight              | Number / String   | -                                                 | 表格最大高度，超出后会出现滚动条。示例：`100, '300'`。值为数字类型，会自动加上单位 px                                                                                                                                                                                                                               | N    |
| loading                | Boolean / Object  | -                                                 | 加载中状态。值为 true 会显示默认加载中样式，基于 Loading 组件开发，透传全部 Loading 组件属性。值为 false 则会取消加载状态。TS 类型：`STableProps<T>['loading']`。<a href="./type" target="_blank" rel="noreferrer">类型定义</a>                                                                                     | N    |
| locale                 | Object            | \{filterTitle: '筛选',filterConfirm: '确定',...\} | 国际化配置，默认简体中文，具体使用查看<a href="../guide/locale" target="_blank" rel="noreferrer">国际化</a>，支持全局配置 `STableProvider`。TS 类型：`STableLocale \| STableProps<T>['locale']`。<a href="./type" target="_blank" rel="noreferrer">类型定义</a>                                                     | N    |
| pagination             | Boolean / Object  | -                                                 | 分页配置，当 dataSource 数据长度超过分页大小时，会自动对本地数据 dataSource 进行分页，如果不希望对 dataSource 进行分页，可以设置值为`false`。TS 类型：`false \| Partial<STablePaginationConfig> \| STableProps<T>['pagination']`。<a href="./type" target="_blank" rel="noreferrer">类型定义</a>                    | N    |
| rowClassName           | String / Function | -                                                 | 行类名，泛型 T 指表格数据类型。TS 类型：`STableProps<T>['rowClassName']`。<a href="./type" target="_blank" rel="noreferrer">类型定义</a>                                                                                                                                                                            | N    |

## Column

| className | String            | -      | 表头列类名                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         | N    |
